# sim/ — PL CAN / 반사 로직 시뮬레이션 (시뮬-퍼스트)

> RTL을 **합성 없이 초 단위로 반복**하는 곳. 반사 로직 시행착오는 여기서 끝내고,
> 합성/HIL은 통과한 것만. (PL 빠른 반복의 핵심)

## 06에서 시뮬할 것
- CTU CAN-FD(또는 커스텀 CAN TX FSM)에 **CAN 프레임 자극** 넣고 TX 파형 확인.
- 1Mbps 비트타이밍, 0x155 같은 프레임이 버스에 맞게 나가는지.

## 07+에서 시뮬할 것 (반사)
- DIP 트리거 → MUX가 PS명령 차단 → 반사 FSM이 0x150(e-stop) 주입하는 시퀀스.
- **프레임 경계 전환**(중간 차단 시 에러프레임) 검증.

## 도구 (택1)
- **Vivado xsim** — 블록디자인/IP 그대로 시뮬, 통합 쉬움.
- **Verilator** — 빠르고 CI 친화적, C++ 테스트벤치.

## 구성 (채워나갈 것)
```
sim/
├── tb_*.v / tb_*.sv     테스트벤치 (CAN 프레임 자극)
├── stim/                자극 벡터 (예: 0x155 명령, 0x150 e-stop)
└── run_sim.sh           xsim/verilator 실행 (만들 것)
```
> 반사 RTL 모듈은 07+(`02_reflex_system_dev`)에서 _v숫자로 버전관리, 여기 sim으로 먼저 검증.
